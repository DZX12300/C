复杂条件构造：

1.数值比较：比较两个数的大小值。

2.字符的比较：比较两个字符对应的ASCII值的大小。

'a' > 'b'    0          97  >  98

ASCII规律：

0~9   48~57

A~Z  65~90

a~z   97~122

3.两数同号：  a * b > 0

4.a,b,c 不全为0： a || b || c != 0
5.x为整数：x - (int)x == 0

6.x不为整数：x - (int)x != 0

7.四位正整数n为闰年：

闰年：四年一闰，百年不闰，四百年再闰

（n % 4 == 0 && n % 100 != 0）||（n % 4 == 0 && n % 400 == 0）

   x % 3 == 0 && x % 7 == 0
